## 📋 How to use the software
When you open the app, you see a main menu. This menu shows you the available tools. Choose a folder from the library to see the skills inside. Each skill performs a specific task, such as creating a database or setting up a user login screen.

Drag the skills you need into the main project window. Connect the skills in the order you want the software to run. Once you place the items, click the Generate button. The system writes the logic for you. You can review the finished structure before you save it to your computer.
